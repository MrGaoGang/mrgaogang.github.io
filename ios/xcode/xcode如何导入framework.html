    # xcode 如何导入静态 framework

    1. 在工程根目录下新建一个Frameworks目录

    2. 在设置 Framework 搜索路径

    3. 在将需要导入的 framework 拷贝到上述创建的Frameworks目录中

    4. 在Build Phases中选择Embed Frameworks中新增，并选择其他，选择刚copy进去的framework.
