# docker 缓存 node_modules的方案

# 方案1. 复制package.json的方式

整体思路:

  • 将项目的package.json复制到docker的安装目录下
  • 切换到安装目录，执行npm install
  • 复制目录下的node_modules 到应用的目录下
COPY app/package.json /opt/app/package.json
RUN cd /opt/app && npm install
COPY app /opt/app
